$BUILD Token
$BUILD token is used as the governance token for the DAO.

It represents a pro-rata claim of ownership on all DAO’s revenues, assets and liabilities (incl. BUILD Treasury and $bCRED debt token).

BUILD currently runs a staking programme to incentivise Uniswap liquidity providers for BUILD/ETH pair.
